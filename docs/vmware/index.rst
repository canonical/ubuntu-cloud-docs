Ubuntu on VMware
================

**Ubuntu on VMware is a set of customized Ubuntu images** that allow easy access to a wide range
of products and services - offered by both VMware and Canonical.

**These images provide a foundation for deploying cloud-based software solutions,** specifically
for software built on Ubuntu and running on VMware. They focus on providing the optimal tools
and features needed to run specific workloads.
