Support options on Azure
========================


Ubuntu Pro
----------

`Ubuntu Pro with Support`_ outlines the support options available on Azure. Ubuntu Pro purchased through the Azure
marketplace does not include 24x7 SLA-backed support by default. Additional support is available through either:

* Private offers
* Ubuntu Pro tokens

Contact azure@ubuntu.com for additional support options on Azure.

An Ubuntu Pro subscription also provides access to our `Knowledge Base`_ which is a repository of technical documentation, how-to-guides, and tutorials on Canonical products.


Azure support
-------------

Technical support for Azure is available for those with a `support plan`_.


Public Bug Tracker
------------------

The Canonical Public Cloud (CPC) team maintains a bug tracker for all cloud images. If you find
a problem with a cloud image you can `file a bug`_.

When creating a bug:

* Describe the issue
* Provide a reproducer
* Include the Azure marketplace image URN (``publisher:offerID:planID:imageVersion``) that is affected

While CPC regularly triage bugs, support here is best effort.

.. _`file a bug`: https://bugs.launchpad.net/cloud-images/+filebug/+login
.. _`Knowledge base`: https://documentation.ubuntu.com/pro/support-overview/#knowledge-base
.. _`Ubuntu Pro with Support`: https://ubuntu.com/azure/support
